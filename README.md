# Webkelas: Big Family XII A1
Webkelas adalah aplikasi web modern untuk angkatan **Big Family XII A1**, menampilkan galeri, chatting, dan fitur lainnya.

## Fitur Utama
- UI modern dengan Glassmorphism & Framer Motion.
- Jadwal kelas & Galeri foto.
- Admin Dashboard (Akses via `#/admin`).
- Mode Gelap/Terang otomatis.

## Cara Menjalankan
1. Clone repo:
   ```bash
   git clone https://github.com/Har404-err/Webkelas.git
   ```
2. Install & Run:
   ```bash
   npm install
   npm run dev
   ```
